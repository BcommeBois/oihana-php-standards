## Vue d'ensemble

`ISO15924` énumère les codes ISO 15924 pour les **systèmes d'écriture** (scripts). Les valeurs sont en **Titlecase** (`Latn`, `Cyrl`, `Hant`, etc.), 4 lettres conformément au standard.

Inclut aussi des codes spéciaux (privé, indéterminé) :
- `Zinh` — script hérité
- `Zyyy` — script commun (Unicode)
- `Zzzz` — script non codé
- `Qaaa`–`Qabx` — usage privé

## Exemples

```php
use org\iso\ISO15924;

ISO15924::LATN;                   // "Latn"
ISO15924::includes('Hant');       // true
ISO15924::includes('Wxyz');       // false
ISO15924::enums();                // ['Adlm', ..., 'Zzzz']
ISO15924::getConstant('Cyrl');    // "CYRL"
```

### Cas d'usage — composer un tag BCP 47

```php
use org\ietf\Locale;
use org\iso\ISO15924;
use org\iso\ISO639_1;
use org\iso\ISO3166_1;

// Construire "zh-Hant-TW"
$tag = ISO639_1::ZH . '-' . ISO15924::HANT . '-' . ISO3166_1::TW;
new Locale($tag, strict: true); // OK
```
