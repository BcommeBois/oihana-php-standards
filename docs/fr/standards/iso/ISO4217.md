## Vue d'ensemble

`ISO4217` énumère les codes monnaies alpha-3 ISO 4217 (`USD`, `EUR`, `JPY`, etc.). Le nom de la constante est identique à la valeur (ex. `ISO4217::USD === 'USD'`).

## Exemples

```php
use org\iso\ISO4217;

ISO4217::USD;                  // "USD"
ISO4217::includes('EUR');      // true
ISO4217::includes('XYZ');      // false
ISO4217::validate('JPY');      // 'JPY'
ISO4217::validate('FAKE');     // throws ConstantException
ISO4217::enums();              // ['AED', 'AFN', ..., 'ZWL']
ISO4217::getConstant('GBP');   // "GBP"
```

### Cas d'usage métier — validation d'une saisie de devise

```php
use org\iso\ISO4217;
use InvalidArgumentException;

function validateCurrency(string $code): string
{
    $code = strtoupper($code);
    if (!ISO4217::includes($code))
    {
        throw new InvalidArgumentException("Code monnaie inconnu : $code");
    }
    return $code;
}

validateCurrency('eur'); // "EUR"
validateCurrency('xyz'); // throws
```
