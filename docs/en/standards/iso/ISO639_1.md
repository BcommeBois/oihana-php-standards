## Overview

`ISO639_1` enumerates ISO 639-1 alpha-2 language codes (`fr`, `en`, `de`, etc.). Values are **lowercase**, per the standard.

## Examples

```php
use org\iso\ISO639_1;

ISO639_1::EN;                   // "en"
ISO639_1::includes('fr');       // true
ISO639_1::includes('ZZ');       // false (the class stores in lowercase)
ISO639_1::enums();              // ['aa', 'ab', ..., 'zu']
ISO639_1::getConstant('ja');    // "JA"
```

### Domain use case â€” language negotiation with the `Accept-Language` header

```php
use org\iso\ISO639_1;

function pickSupportedLang(string $acceptLanguage, array $supported): string
{
    foreach (explode(',', $acceptLanguage) as $part)
    {
        $code = strtolower(trim(explode(';', $part)[0]));
        $code = substr($code, 0, 2); // keep only language, ignore region
        if (ISO639_1::includes($code) && in_array($code, $supported, true))
        {
            return $code;
        }
    }
    return $supported[0];
}

pickSupportedLang('fr-FR, en;q=0.8', ['en', 'fr']); // "fr"
```
