## Overview

`ISO15924` enumerates ISO 15924 codes for **writing systems** (scripts). Values are in **Titlecase** (`Latn`, `Cyrl`, `Hant`, etc.), 4 letters per the standard.

Also includes special codes (private, undetermined):
- `Zinh` — inherited script
- `Zyyy` — common script (Unicode)
- `Zzzz` — uncoded script
- `Qaaa`–`Qabx` — private use

## Examples

```php
use org\iso\ISO15924;

ISO15924::LATN;                   // "Latn"
ISO15924::includes('Hant');       // true
ISO15924::includes('Wxyz');       // false
ISO15924::enums();                // ['Adlm', ..., 'Zzzz']
ISO15924::getConstant('Cyrl');    // "CYRL"
```

### Use case — composing a BCP 47 tag

```php
use org\ietf\Locale;
use org\iso\ISO15924;
use org\iso\ISO639_1;
use org\iso\ISO3166_1;

// Build "zh-Hant-TW"
$tag = ISO639_1::ZH . '-' . ISO15924::HANT . '-' . ISO3166_1::TW;
new Locale($tag, strict: true); // OK
```
