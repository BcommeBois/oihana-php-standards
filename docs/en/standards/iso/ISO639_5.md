## Overview

`ISO639_5` enumerates alpha-3 codes for **language families and groups** — as opposed to individual languages covered by [`ISO639_1`](ISO639_1.md), [`ISO639_2`](ISO639_2.md), etc.

Categories covered:

- **Language families**: `roa` Romance, `gem` Germanic, `sla` Slavic, `cel` Celtic
- **Areal / geographic groupings**: `aus` Australian, `nai` North American Indian
- **Language phyla**: `afa` Afro-Asiatic, `nic` Niger-Congo

Useful for:
- **Language fallback chains** (if French `fr` is unavailable, fall back to another Romance language `roa`)
- **Bibliographic / linguistic** classification

> ℹ️ ~65 ISO 639-5 codes are also present in [`ISO639_2`](ISO639_2.md) — these codes were historically assigned to families in ISO 639-2 before ISO 639-5 was formalized. Each registry remains independent and authoritative for its intended use.

## Examples

```php
use org\iso\ISO639_5;

ISO639_5::ROA;                  // 'roa' (Romance)
ISO639_5::GEM;                  // 'gem' (Germanic)
ISO639_5::includes('sla');      // true
ISO639_5::includes('fra');      // false (individual language → ISO639_2)
ISO639_5::getConstant('cel');   // 'CEL'
```

### Use case — language fallback chain

```php
use org\iso\ISO639_1;
use org\iso\ISO639_5;

/**
 * Simplified language → family mapping.
 */
function languageFamily( string $lang ): ?string
{
    return match( $lang )
    {
        'fr', 'es', 'it', 'pt', 'ro', 'ca' => ISO639_5::ROA,  // Romance
        'en', 'de', 'nl', 'sv', 'da', 'no' => ISO639_5::GEM,  // Germanic
        'ru', 'pl', 'cs', 'sk', 'bg'       => ISO639_5::SLA,  // Slavic
        default => null,
    };
}

function resolveContent( string $userLang, array $available ): ?string
{
    if ( in_array($userLang, $available, true) )
    {
        return $userLang;
    }

    // Try another language from the same family
    $family = languageFamily($userLang);
    foreach ( $available as $candidate )
    {
        if ( languageFamily($candidate) === $family )
        {
            return $candidate;
        }
    }

    return null;
}

resolveContent('fr', ['it', 'de']);   // 'it' (same Romance family)
resolveContent('da', ['nl', 'fr']);   // 'nl' (same Germanic family)
```
