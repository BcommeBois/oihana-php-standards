## Overview

`UNM49Numeric` enumerates the **3-digit numeric codes** of the UN M49 standard published by the United Nations Statistics Division. The standard covers:

- **countries and areas** (overlap with ISO 3166-1 numeric) — e.g. `004` Afghanistan, `250` France, `840` United States of America;
- **geographical groupings** (continents, sub-regions, special categories) — e.g. `001` World, `019` Americas, `142` Asia, `150` Europe, `419` Latin America and the Caribbean.

Values always preserve the **3-digit zero-padded** form (`'004'`, not `'4'`).

> For alpha-3 codes (`FRA`, `USA`, …), see [`UNM49`](UNM49.md). For alpha-2 codes (`FR`, `US`, …), see [`ISO3166_1`](../iso/ISO3166_1.md).

## Naming convention

PHP identifiers cannot start with a digit, so **all constants are prefixed with `M_`**:

| Constant | Value | Description |
|---|---|---|
| `M_001` | `'001'` | World (M49 region) |
| `M_004` | `'004'` | Afghanistan |
| `M_250` | `'250'` | France |
| `M_419` | `'419'` | Latin America and the Caribbean (M49 region) |

This is consistent with the convention used for numeric BCP 47 variants (`V_1996`, etc.).

## M49 regions (BCP 47-compatible subset)

According to [RFC 5646 §2.2.4](https://www.rfc-editor.org/rfc/rfc5646#section-2.2.4), only **M49 region codes that do not also have an ISO 3166-1 alpha-2 equivalent** are allowed as numeric `region` subtags:

| `002` | Africa |
| `019` | Americas |
| `009` | Oceania |

| `150` | Europe |
| `419` | Latin America and the Caribbean |
| `029` | Caribbean |
| `030` | Eastern Asia |
| `034` | Southern Asia |
| `035` | South-eastern Asia |
| `143` | Central Asia |
| `145` | Western Asia |

| `011` | Western Africa |

## Examples

```php
use org\unstats\UNM49Numeric;

UNM49Numeric::M_250;               // '250' (France)
UNM49Numeric::M_419;               // '419' (Latin America and the Caribbean)

UNM49Numeric::includes('001');     // true
UNM49Numeric::includes('999');     // false (unassigned)

UNM49Numeric::getConstant('419');  // 'M_419'
```

### Use case — regional BCP 47 locale

```php
use org\ietf\Locale;

// Latin American Spanish (region subtag = 419)
$locale = new Locale('es-419', strict: true);
echo $locale->language;  // 'es'
echo $locale->region;    // '419'
```

In strict mode, [`isLocale`](../ietf/helpers/isLocale.md) cross-validates 3-digit regions against `UNM49Numeric`:

```php
use function org\ietf\helpers\isLocale;

isLocale('es-419', strict: true);   // true  (419 = Latin America and the Caribbean)
isLocale('es-999', strict: true);   // false (999 unassigned)
```

