## Overview

`ISO3166_1` enumerates ISO 3166-1 alpha-2 country codes (`FR`, `US`, `JP`, etc.). Values are **uppercase**, per the standard.

> For alpha-3 and numeric codes (UN M49), see [`UNM49`](../unstats/UNM49.md).

## Examples

```php
use org\iso\ISO3166_1;

ISO3166_1::FR;                   // "FR"
ISO3166_1::includes('US');       // true
ISO3166_1::includes('XX');       // false
ISO3166_1::validate('DE');       // 'DE'
ISO3166_1::enums();              // ['AD', 'AE', ..., 'ZW']
ISO3166_1::getConstant('JP');    // "JP"
```

### Domain use case — country validation in a form

```php
use org\iso\ISO3166_1;
use InvalidArgumentException;

function validateCountry(string $code): string
{
    $code = strtoupper($code);
    if (!ISO3166_1::includes($code))
    {
        throw new InvalidArgumentException("Unknown country code: $code");
    }
    return $code;
}

validateCountry('fr'); // "FR"
```
