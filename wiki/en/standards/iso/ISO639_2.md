## Overview

`ISO639_2` enumerates the **alpha-3** ISO 639-2 language codes in their **canonical form**:

- the **terminologic (T)** form when a language has both a bibliographic (B) code and a terminologic code (`fra`, `deu`, `zho`, `nld`, `ell`, `isl`, `ron`, etc.);
- the single alpha-3 code otherwise (`eng`, `spa`, `ita`, etc.).

This preference for the **T** form follows [RFC 5646 В§4.1.2](https://www.rfc-editor.org/rfc/rfc5646#section-4.1.2) (IETF BCP 47).

> For **bibliographic (B)** codes only (~20 codes: `fre`, `ger`, `chi`, etc.) and Bв†’T conversion, see [`ISO639_2B`](ISO639_2B.md).
> For alpha-2 codes (subset of ~184 languages), see [`ISO639_1`](ISO639_1.md).
> For language families (`roa` Romance, `gem` GermanicвЂ¦), see [`ISO639_5`](ISO639_5.md).

## Examples

```php
use org\iso\ISO639_2;

ISO639_2::FRA;                  // 'fra'
ISO639_2::DEU;                  // 'deu'
ISO639_2::includes('eng');      // true
ISO639_2::includes('fre');      // false вЂ” 'fre' is bibliographic (see ISO639_2B)
ISO639_2::includes('zzz');      // false
ISO639_2::getConstant('jpn');   // 'JPN'
```

### Use case вЂ” language without ISO 639-1 equivalent

```php
use org\iso\ISO639_1;
use org\iso\ISO639_2;

// 'kal' (Greenlandic / Kalaallisut) has no ISO 639-1 alpha-2 code
$language = 'kal';
ISO639_1::includes($language);  // false
ISO639_2::includes($language);  // true
```

### Use case вЂ” BCP 47 locale with 3-letter language

```php
use org\ietf\Locale;

// 'haw' (Hawaiian) вЂ” alpha-3 only
$locale = new Locale('haw-US', strict: true);
echo $locale->language;  // 'haw'
echo $locale->region;    // 'US'
```
