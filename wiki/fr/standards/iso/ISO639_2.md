## Vue d'ensemble

`ISO639_2` énumère les codes langue **alpha-3** ISO 639-2 sous leur **forme canonique** :

- la forme **terminologique (T)** quand la langue possède à la fois un code bibliographique (B) et un code terminologique (`fra`, `deu`, `zho`, `nld`, `ell`, `isl`, `ron`, etc.) ;
- la forme alpha-3 unique sinon (`eng`, `spa`, `ita`, etc.).

Cette préférence pour la forme **T** est celle prescrite par [RFC 5646 §4.1.2](https://www.rfc-editor.org/rfc/rfc5646#section-4.1.2) (IETF BCP 47).

> Pour les codes **bibliographiques (B)** uniquement (~20 codes : `fre`, `ger`, `chi`, etc.) et la conversion B→T, voir [`ISO639_2B`](ISO639_2B.md).
> Pour les codes alpha-2 (sous-ensemble de ~184 langues), voir [`ISO639_1`](ISO639_1.md).
> Pour les familles linguistiques (`roa` Romance, `gem` Germanique…), voir [`ISO639_5`](ISO639_5.md).

## Codes spéciaux inclus

| Constante | Code | Description |
|---|---|---|
| `MIS` | `mis` | Uncoded languages (langues non codées) |
| `MUL` | `mul` | Multiple languages (langues multiples) |
| `UND` | `und` | Undetermined (indéterminé) — code par défaut de [`Locale::ZERO`](../ietf/Locale.md) |
| `ZXX` | `zxx` | No linguistic content (pas de contenu linguistique) |

> 🚫 La plage `qaa-qtz` (réservée à l'usage local) **n'est volontairement pas énumérée** : ces codes ne sont pas individuellement assignés par l'ISO et doivent être définis localement par l'utilisateur (même approche que `Qaaa`–`Qabx` dans [`ISO15924`](ISO15924.md)).

## Exemples

```php
use org\iso\ISO639_2;

ISO639_2::FRA;                  // 'fra'
ISO639_2::DEU;                  // 'deu'
ISO639_2::includes('eng');      // true
ISO639_2::includes('fre');      // false — 'fre' est bibliographique (voir ISO639_2B)
ISO639_2::includes('zzz');      // false
ISO639_2::getConstant('jpn');   // 'JPN'
```

### Cas d'usage — langue sans équivalent ISO 639-1

```php
use org\iso\ISO639_1;
use org\iso\ISO639_2;

// 'kal' (Greenlandic / Kalaallisut) n'a pas de code alpha-2 ISO 639-1
$language = 'kal';
ISO639_1::includes($language);  // false
ISO639_2::includes($language);  // true
```

### Cas d'usage — locale BCP 47 avec langue 3 lettres

```php
use org\ietf\Locale;

// 'haw' (Hawaiian) — alpha-3 uniquement
$locale = new Locale('haw-US', strict: true);
echo $locale->language;  // 'haw'
echo $locale->region;    // 'US'
```
