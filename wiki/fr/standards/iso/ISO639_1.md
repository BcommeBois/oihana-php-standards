## Vue d'ensemble

`ISO639_1` énumère les codes langue alpha-2 ISO 639-1 (`fr`, `en`, `de`, etc.). Les valeurs sont **en minuscules**, conformément au standard.

## Exemples

```php
use org\iso\ISO639_1;

ISO639_1::EN;                   // "en"
ISO639_1::includes('fr');       // true
ISO639_1::includes('ZZ');       // false (la classe stocke en minuscule)
ISO639_1::enums();              // ['aa', 'ab', ..., 'zu']
ISO639_1::getConstant('ja');    // "JA"
```

### Cas d'usage métier — négociation de langue avec en-tête `Accept-Language`

```php
use org\iso\ISO639_1;

function pickSupportedLang(string $acceptLanguage, array $supported): string
{
    foreach (explode(',', $acceptLanguage) as $part)
    {
        $code = strtolower(trim(explode(';', $part)[0]));
        $code = substr($code, 0, 2); // garde uniquement la langue, ignore la région
        if (ISO639_1::includes($code) && in_array($code, $supported, true))
        {
            return $code;
        }
    }
    return $supported[0];
}

pickSupportedLang('fr-FR, en;q=0.8', ['en', 'fr']); // "fr"
```
