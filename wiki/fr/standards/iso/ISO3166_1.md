## Vue d'ensemble

`ISO3166_1` énumère les codes pays alpha-2 ISO 3166-1 (`FR`, `US`, `JP`, etc.). Les valeurs sont **en majuscules**, conformément au standard.

> Pour les codes alpha-3 et numériques (UN M49), voir [`UNM49`](../unstats/UNM49.md).

## Exemples

```php
use org\iso\ISO3166_1;

ISO3166_1::FR;                   // "FR"
ISO3166_1::includes('US');       // true
ISO3166_1::includes('XX');       // false
ISO3166_1::validate('DE');       // 'DE'
ISO3166_1::enums();              // ['AD', 'AE', ..., 'ZW']
ISO3166_1::getConstant('JP');    // "JP"
```

### Cas d'usage métier — validation d'un pays dans un formulaire

```php
use org\iso\ISO3166_1;
use InvalidArgumentException;

function validateCountry(string $code): string
{
    $code = strtoupper($code);
    if (!ISO3166_1::includes($code))
    {
        throw new InvalidArgumentException("Code pays inconnu : $code");
    }
    return $code;
}

validateCountry('fr'); // "FR"
```
